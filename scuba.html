Step 1: Download SCUBA

SCUBA (Security Compliance Baseline Analyzer) is a tool used to inspect security configurations for Microsoft Office 365 and Azure.

Download the latest version of SCUBA from the official DISA website.

Step 2: Install SCUBA

After downloading the SCUBA installer:

  1. Locate the downloaded .zip file.
  2. Extract the contents to a secure folder.
  3. Run the setup.exe file to begin installation.
  4. Follow the on-screen instructions to complete the setup.

Step 3: Launch SCUBA

Once installed, open SCUBA by:

  1. Clicking on the SCUBA desktop shortcut or searching for "SCUBA" in the start menu.
  2. Running it with administrator privileges to access full functionality.

Step 4: Configure SCUBA for Office 365 and Azure

To inspect Microsoft Office 365 and Azure security configurations:

  1. Open SCUBA and navigate to the "Settings" section.
  2. Select "Microsoft Office 365 & Azure" from the compliance options.
  3. Enter your administrator credentials to connect SCUBA to your environment.
  4. Customize security benchmarks if needed.

Step 5: Run a Security Scan

To analyze your Office 365 and Azure configurations:

  1. Go to the "Scan" section.
  2. Click on "Start Scan" and select Office 365 or Azure.
  3. Wait for the analysis to complete.
  4. Review the detailed security assessment report.

Step 6: Review and Address Security Findings

SCUBA generates a report with security compliance recommendations:

  1. Navigate to the "Reports" tab.
  2. Identify security vulnerabilities and misconfigurations.
  3. Follow the suggested remediation steps.
  4. Re-run SCUBA to ensure compliance improvements.

Step 7: Schedule Regular Audits

To maintain security compliance:

  • Set up scheduled scans in SCUBA.
  • Monitor security changes in Office 365 and Azure.
  • Update SCUBA regularly for the latest security policies.

Conclusion

By following these steps, you can effectively use SCUBA to inspect and enhance the security of your Microsoft Office 365 and Azure environments.
