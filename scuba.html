Step 1: Verify PowerShell Version

SCUBA requires PowerShell 5.1 to function correctly. To check your PowerShell version:

powershell $PSVersionTable.PSVersion

If the major version is less than 5.1, you need to update PowerShell.

Step 2: Install PowerShell 5.1 (If Needed)

If you are running an older version, follow these steps to install PowerShell 5.1:

  1. Download the Windows Management Framework 5.1.
  2. Run the downloaded .msu installer.
  3. Restart your computer after installation.
  4. Re-run $PSVersionTable.PSVersion to confirm the update.

Step 3: Install SCUBA Using PowerShell 5.1

Once the SCUBA installer is downloaded, follow these steps to install it via PowerShell:

  1. Open PowerShell 5.1 as Administrator.
  2. Run the installer:
    3. Set-ExecutionPolicy Unrestricted
    Install-Module -Name ScubaGear -Force
    Initialize-SCuBA
    Invoke-SCuBA

Step 4: Review and Address Security Findings

SCUBA generates a report with security recommendations:

  1. Navigate to the "Reports" tab.
  2. Identify vulnerabilities and misconfigurations.
  3. Follow the recommended remediation steps.
  4. Re-run SCUBA to verify compliance improvements.

Conclusion

By following these steps, you can effectively use SCUBA to inspect and enhance the security of your Microsoft Office 365 and Azure environments.
