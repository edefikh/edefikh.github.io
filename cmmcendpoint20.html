SOC 2 & CMMC 2.0 Endpoint Security Checklist

1. User Account & Access Management

  • Disable local admin accounts
  • Enforce least privilege access
  • Enable and enforce Multi-Factor Authentication (MFA)
  • Disable Guest Accounts
  • Enable FileVault encryption (Mac)

2. Device Hardening & Security Configurations

  • Enable BitLocker (Windows) or FileVault (Mac)
  • Disable Remote Desktop Protocol (RDP) if not needed
  • Enforce strong password policies
  • Disable USB ports for unauthorized devices

3. Endpoint Monitoring & Logging

  • Enable Windows Event Logging
  • Configure audit policies
  • Enable macOS Unified Logging
  • Monitor for unauthorized PowerShell scripts

4. Patch Management & System Updates

  • Enable automatic updates for OS & applications
  • Apply critical security patches immediately
  • Update third-party software regularly

5. Application Security & Control

  • Remove unnecessary applications
  • Enable AppLocker or WDAC (Windows)
  • Allow only signed applications (Mac)
  • Block macros in Office documents

6. Network Security & Remote Access

  • Enable Windows Defender Firewall
  • Disable Bluetooth & Wi-Fi auto-connect
  • Restrict VPN access to authorized users
  • Use DNS filtering solutions

7. Endpoint Detection & Response (EDR)

  • Enable Windows Defender ATP or third-party EDR
  • Configure real-time threat protection
  • Enable tamper protection

8. Data Protection & Backup

  • Enable OneDrive or Time Machine backups
  • Enforce Data Loss Prevention (DLP) policies
  • Store backups securely in multiple locations

9. Incident Response & Reporting

  • Define incident response procedures
  • Configure alerts for security incidents
  • Regularly test incident response plans

10. Compliance & Audit Readiness

  • Maintain audit logs for at least 12 months
  • Perform regular security assessments
  • Ensure all configurations align with SOC 2 & CMMC 2.0
